Description

The form is divided into individual pages.
Each page has a it's own background, name and multiple properties.
Each property has multiple options - either text based or swatch based.
Lastly, each option has a name, price, an overlay image and possibly a swatch image (if it is a swatch based option) associated with it. The overlay image will be laid over the page background when the option is selected in the form.

The current form is automatically saved to the browser's local storage, so leaving the site shouldn't delete your progress. Note that shutting off your computer, closing the browser completely or even accessing some other sites might clear this data and it'll be lost. It is recommended to export and save the code as you go.

All fields are absolutely mandatory for the code to work properly! (there is only one exception - the field called 'Item description' is optional)
